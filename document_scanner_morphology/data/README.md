# 数据集准备说明

本项目不依赖公开数据集，可使用自建手机拍摄文档图像集。

## 建议数量

建议至少拍摄 20 张图像，文件命名为：

```text
doc_001.jpg
doc_002.jpg
doc_003.jpg
...
```

## 图像类型

建议包含以下类型：

- 普通 A4 文档
- 实验报告
- 试卷
- 书本页面
- 有阴影的文档
- 倾斜拍摄的文档
- 背景复杂的文档

## 采集条件

- 使用手机拍摄
- 分辨率不限
- 包含不同光照条件
- 包含不同倾斜角度
- 尽量保留完整文档边界

## 标注方式

本项目不需要像素级人工标注。实验时可人工记录文档角点检测是否成功，并主观观察固定阈值、Otsu 和 Sauvola 三种二值化结果的清晰度、断笔和噪声情况。
